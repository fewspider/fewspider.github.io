一直想给blog加个缩放图片的效果，以前在wordpress用习惯了Auto HighSlide（貌似就这名字），效果很赞。 搬家到jekyll后，自然不能少了缩放图片这一功能。

fancyBox是一款基于jquery开发的类Lightbox插件。支持对放大的图片添加阴影效果，对于一组相关的图片添加导航操作按纽，该lightbox除了能够展示图片之外，还可以展示iframed内容， 通过css自定义外观。

到 fancyBox 主页下载源码，解压，找到 jquery.fancybox.css、jquery.fancybox.js和jquery.mousewheel-3.0.6.pack.js,放在自己的jekyll站点目录，路径参考别的css和js，放在一起，比较方便嘛。
在主题模板文件<head></head>标签插入刚才的js和css文件，路径找好，不一定是我示例这路径。 

<link href="{{ ASSET_PATH }}/css/jquery.fancybox.css?v=2.1.5" rel="stylesheet" />
<script type="text/javascript" src="{{ ASSET_PATH }}/js/jquery.mousewheel-3.0.6.pack.js"></script>
<script type="text/javascript" src="{{ ASSET_PATH }}/js/jquery.fancybox.js?v=2.1.5"></script>

因为fancyBox是基于jquery开发的，所以记得要加载jquery库。jquery可以到 jquery 下载，添加下面一行，路径根据自己的情况： 

<script type="text/javascript" src="{{ ASSET_PATH }}/js/jquery-1.8.2.min.js"></script>

除了这些，还有jquery.fancybox.css里面的图片，也要把她复制到你自己的文件夹，别忘了修改路径。 比如 

.fancybox-overlay {
	position: absolute;
	top: 0;
	left: 0;
	overflow: hidden;
	display: none;
	z-index: 8010;
	background: url('../bootstrap/img/fancybox_overlay.png');
}
其中的background路径，我就根据自己的情况作了修改。

在模板文件head标签处还要插入一行代码： 

<script type="text/javascript">
		$(document).ready(function() {
		$('.fancybox').fancybox();
		});
	</script>

没出什么错误的话，更新下jekyll，图片缩放的效果就出来了。除了点击按钮，还可以通过滑动鼠标，进行下一张图片的预览，这个蛮帅的。
