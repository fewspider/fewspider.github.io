--- layout: post category : linux title: "解决Amarok播放列表文件名乱码" date: 2013-10-23 19:32:37 +0800 tags : [Amarok] --- 最近导入mp3到Amarok，播放列表经常乱码，看着好心烦，毕竟咱不是火星人，估计火星人也看不懂吧。 解决乱码的办法如下 安装python-mutagen： 
yum install python-mutagen
当前目录： 
mid3iconv -e GBK *.mp3
包含子目录： 
mid3iconv -e GBK */*.mp3
删掉刚才的乱码播放列表，重新导入，就没有乱码了。
