--- layout: post category : linux title: "Fedora19下安装Fcitx-mozc" date: 2013-07-27 15:18:11 -0700 tags : [fedora , fcitx ] --- 刚开始接触Fedora的时候，还不大熟悉。最近比较顺手了，就是挺纠结Fcitx不能支持日语。早上在推上抱怨了几句，然后作者就为Fedora 19更新了（偷笑。 以下是针对Fedora 19的安装办法，以`root`权限运行 
cd /etc/yum.repos.d/
wget http://download.opensuse.org/repositories/home:csslayer:fedora-mozc/Fedora_19/home:csslayer:fedora-mozc.repo
yum install fcitx-mozc
这样就安装好了，默认是有添加到当前输入法的，切换下一输入法快捷键默认是Ctrl+Shift。
