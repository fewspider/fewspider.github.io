--- layout: post category : lessons title: "解决myeclipse无法打开jsp文件" date: 2014-07-07 13:18:35 tags : [ myeclipse , jsp ] ---

在处理jsp的一些页面，myeclipse工具下编辑的，可以跑起来。可是把myeclipse关掉后，重新打开刚才那个jsp文件，myeclipse就报错了。提示：Failed to create the part's controls

刚开始以为是自己敲的代码有问题，后来发现不是= =。挺倒霉的其实，myeclipse里面写jsp页面，上面有一个预览，类似Dreamweaver里面的设计视图。╮(￣▽￣")╭当然愿望是美好的，myeclipse里面的那个设计视图烂得不能再烂。问题其实就出现在这边。

不是自己的代码有问题，而是myeclipse自己的bug，页面写的代码有点多，设计视图就崩溃了。把它关掉，就没问题了。

window -> perferences -> General -> editors -> file associations 找到jsp -> 把 myeclispe jsp editor 设为 default（默认是：myeclispe visual jsp editor）

这样就没问题啦φ(*￣0￣)>
