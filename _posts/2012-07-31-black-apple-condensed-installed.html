黑苹果，想必大家都知道了吧，就是在普通pc上安装苹果的系统，小编折腾的是OS X Mountain Lion。

理论上，安装没什么问题，就是驱动难找了点，目前就网卡、声卡完美驱动，显卡还在头疼。

先晒晒电脑的配置：

          电脑型号 华硕 N53SM 笔记本电脑
          处理器 英特尔 Core i5-2450M @ 2.50GHz 双核
          主板 华硕 N53SM (英特尔 HM65 芯片组)
          内存 4 GB ( 海力士 DDR3 1333MHz )
          主硬盘 希捷 ST9750420AS ( 750 GB / 7200 转/分 )
          显卡 Nvidia GeForce GT 630M ( 2 GB / 华硕 )
          显示器 奇美 CMO15A7 ( 15.7 英寸 )
          光驱 建兴 DVD A DS8A8SH DVD刻录机
          声卡 瑞昱 ALC269 @ 英特尔 6 Series Chipset 高保真音频
          网卡 瑞昱 RTL8168E PCI-E Gigabit Ethernet NIC / 华硕

我用的是硬盘安装，还可以用U盘安装，光盘安装，当然虚拟机也不错，不用担心驱动的问题，就是内存比较吃紧，这里不详细介绍，省略六十万字。

前提条件：

          BIOS开启AHCI
          CPU:Intel Core 2 Duo 以上
          dmg空间大于4.54GB
          安装空间大于6.86GB

小编弄的是GM版本的，跟正式版一样。

          ML安装包及补丁-下载：http://115.com/file/anb4sllw#
          ML安装工具-下载：http://115.com/file/dpsyq3wb#
          变色龙-下载：http://115.com/file/anb4qqvs#
          Extra-下载：http://115.com/file/e72h7lpx#

1.分区，右键计算机，管理，磁盘管理，建立两个分区： 一个分区用来引导安装,容量3g 一个分区就是你安装后的系统盘 再腾出一些可用空间，用来扩充那个3g的分区，容量至少2g 具体步骤：新建简单卷，调整卷大小，不要格式化这个卷 ps：WIN7系统下会提示格式化窗口，请选择取消 两个分区都这样弄，记住不要格式化!!!!!!!!!

2.好压缩软件打开下载好的磁盘镜像文件，将InstallESD.dmg文件提取出来 路径：Install Mountain Lion\Install OS X Mountain Lion.app\Contents\SharedSupport\InstallESD.dmg

3.用HFSExplorer-0.21打开InstallESD.dmg，并提取/BaseSystem.dmg、mach_kernel及/Packages文件夹 ps：如果无法直接用HFSExplorer打开InstallESD.dmg，用7z软件提取InstallESD.dmg后再操作

4.用HFSExplorer-0.21打开BaseSystem.dmg，并打包建立新的dmg，命名为BaseSystem-HFS.dmg

5.用HD_Install_Helper_v0.3将BaseSystem-HFS.dmg 写入３Ｇ分区 ps：取消在boot.ini中加入tboot、强制加入引导及启动代码、PC_EFI V8三个选项的勾选 当出现--Change partition typpe to AF:Success 提示就表示hfs分区写入成功

6.用 管理员权限 打开6-Paragon.Partition.Man.11.Personal_server\program\launcher.exe 调整 3.0G 的分区大小，扩充为 5.0G

7.用 管理员权限 打开DiskGenius.exe，更改分区参数为AF

8.删除５Ｇ硬盘的/System/Installation/Packages，将提取的/Packages文件夹(共约 3.0 GB)，全部复制到５Ｇ硬盘的/System/Installation/Packages文件夹 ps：这时候还打开不了那些分区，需要安装MacDrive.v8.0.7.38，试用版本，附件中有对应的破解工具

9.把mach_kernel复制到BaseSystem.dmg的５Ｇ硬盘的根目录 / 里

10.替换/System/Installation/Package/OSInstall.mpkg和/System/Library/PrivateFrameworks/Install.framework/Frameworks/OSInstall.framework/Versions/A/OSInstall

11.安装Windows版变色龙引导，替换Windows系统盘根目录wowpc.iso，在mac安装盘手动新建文件夹Extra，按路径放好这些文件 /Extra/org.chameleon.Boot.plist
/Extra/Extensions/*****.kext

12.重启，可以看到除了你之前的系统，下面还出现一个引导条Chameleon，那个就是变色龙，回车进入安装界面，不要急着安装，打开实用工具，磁盘工具，选择mac安装盘，抹掉

          格式：mac扩展日志
          名称：随便，小编的是MAC

折腾好后，退出磁盘工具，接着安装，这时候就可以看到你刚才抹掉的那个盘了，其他盘符有感叹号 选择抹掉的那个盘，安装，耐心等待进度条结束，重启

13.使用PE工具，激活windows安装盘，复制Extra目录到mac盘，重启，进入变色龙引导界面，选择mac盘，就可以进入苹果系统了
