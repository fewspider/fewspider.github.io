--- layout: post category : linux title: "Gnome 3.8 添加应用程序快捷方式" date: 2013-06-26 23:41:33 +0800 tags : [Gnome , fedora , Desktop Entry] --- windows下添加应用程序快捷方式，右键一下，很简单。linux下类似于快捷方式的是Desktop Entry，配置下具体参数，也不难。 小编目前在用的fedora 19,Gnome 版本为 3.8.2。应用程序的快捷方式通常在`/usr/share/applications`这个文件夹，桌面配置文件的后缀名是`.desktop`。 具体参数如下： 
[Desktop Entry]           //每个desktop文件都以这个标签开始，说明这是一个桌面配置文件
Version = 1.0             //标明Desktop Entry的版本（可选）
Name = Firefox            //程序名称（必须），这里以创建一个Firefox的快捷方式为例
GenericName = Web Browser //程序通用名称（可选）
Comment = A Web Browser   //程序描述（可选）
Exec = firefox %u        //程序的启动命令（必选），可以带参数运行
                          //当下面的Type为Application，此项有效
Icon = firefox            //设置快捷方式的图标（可选）
Terminal = false          //是否在终端中运行（可选），当Type为Application，此项有效
Type = Application        //desktop的类型（必选），常见值有“Application”和“Link”
Categories = GNOME;Application;Network; //注明在菜单栏中显示的类别（可选）
`Icon`如果只输入名称，Gnome默认会在`/usr/share/icons`寻找该名称的图标，如果没有，就显示为空白。可以用绝对路径，指出图标文件的位置，例如设置sublime text快捷方式图标。 
Icon = /opt/sublime_text_3/Icon/128x128/sublime-text.png
配置完参数后，你就可以在应用程序那边，找到你自己手动添加的快捷方式啦。
