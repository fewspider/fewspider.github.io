Connectify这个软件用过一段时间，可以分享wifi热点，有破解版的可以无限使用。但是win7系统自带的就可以搞定，没必要安装第三方的软件。而且也比那个软件稳定得多，比较推荐这个。

简单几步

1、开始 键入 cmd ，以管理员权限运行cmd.exe

2、输入以下几行命令

netsh wlan set hostednetwork mode=allow 回车

netsh wlan set hostednetwork ssid=fewspider key=123456789 回车

netsh wlan start hostednetwork 回车

这样一个wifi热点就弄好了，名字为fewspider，密码123456789
      可以自定义，ssid=wifi名称，key=wifi密码** **
      ps:如果提示“无法启动承载网络，组或资源的状态不是执行请求操作的正确状态” ，进入第三步，留下你当前连接网络的那一个，有可能是宽带连接，也有可能是本地连接，把所有连接都禁用，再启用试试，实在不可以就重启系统试试。

3、到 控制面板\网络和 Internet\网络和共享中心 更改适配器设置

右键 宽带连接 修改属性

有可能你的不是无线网络连接2，仔细看你wifi名字在哪一个选项中

这样就可以连接网络了，同样你可以共享vpn，步骤一样的。

怎么关闭wifi呢？

回到第一步，第二步那边键入 netsh wlan stop hostednetwork 这样wifi就关闭了。 每次都要重复这几步来开启或者关闭wifi么？

不用这么麻烦，你可以新建一个txt文本，将那些命令复制过去，保存，改后缀名为bat，右键 以 管理员权限 运行，这样就可以了。

win8也可以这么折腾的，双系统试过，xp可不可以就不清楚了，操作界面有点不同。
