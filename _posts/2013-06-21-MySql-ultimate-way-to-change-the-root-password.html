--- layout: post category : lessons title: "MySql修改root密码终极办法" date: 2013-06-21 13:39:38 +0800 tags : [MySql] --- 最近打算在fedora上面跑MySql，root死活登陆不了。 前前后后安装了n遍，发现还是一样。火了，清除所有MySql残余的文件夹，又安装了n遍，还是那鸟样。正当我打算砸电脑的时候，google搜索里面的一个页面缓缓打开，我又看到了希望。 停用MySql服务： 
service mysqld stop
進入安全模式： 
mysqld_safe --user=mysql --skip-grant-tables --skip-networking &
嘿嘿，這時候就可以不用 root 密碼登入了。 马上更改root密码： 
UPDATE user SET Password=PASSWORD('123456') where USER='root';
quit;
重新启动MySql服务： 
service mysqld restart
试下用root登入： 
mysql -u root -p
输入改才修改的密码，搞定收工。
