--- layout: post category : linux title: "解决Fedora 19多系统时间不一致" date: 2013-06-18 13:06:37 +0800 tags : [fedora] --- 以前安装黑苹果的时候，山猫系统与windows也出现了时间不一致的情况，安装了一个补丁就搞定了。 最近手贱删旧的内核，不小心把ubuntu搞奔溃了，于是干脆换成fedora。也出现了之前类似的情况，系统设置的时区是在同一个地方，可是切换两个系统的话，还是存在时差。小编电脑不止安装了fedora，win7，还安装了win8。win7和win8时间没出现什么问题，自家的嘛。 下面开始解决的办法： 安装ntp： 
sudo yum install ntp ntpdate;
允许ntpd和ntpdate两个服务开机启动： 
sudo systemctl enable ntpd ntpdate;
使用UTC： 
sudo hwclock --systohc --utc;
