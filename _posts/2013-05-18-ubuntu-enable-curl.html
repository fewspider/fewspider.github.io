--- date: 2013-05-18 01:42:15+00:00 layout: post title: Ubuntu apache使用cURL category : linux tags : [Apache, curl, php, ubuntu] --- 1、安装cURL 
sudo apt-get install curl libcurl3 libcurl3-dev php5-curl
2、编辑配置文件 
sudo vi /etc/php5/apache2/php.ini
插入 
extension=curl.so
3、重启apache 
sudo service apache2 restart
